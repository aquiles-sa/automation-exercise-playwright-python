# 🧪 Automation Exercise – Testes Automatizados com Playwright (Python)

Projeto de **automação de testes de interface (UI)** utilizando **Playwright com Python**, aplicado ao site **Automation Exercise**, uma aplicação web de e-commerce usada para prática de testes funcionais end-to-end.

Este projeto tem como objetivo demonstrar conhecimentos fundamentais de **QA**, **automação de testes**, **boas práticas de organização** e **estruturação de testes**.

---

## 🎯 Objetivo do Projeto

- Automatizar fluxos reais de uma aplicação web
- Validar funcionalidades críticas de um e-commerce
- Aplicar boas práticas de testes automatizados
- Criar um projeto de portfólio para nível **estagiário/júnior em QA**

---

## 🌐 Aplicação Testada

- **Site:** https://automationexercise.com  
- **Tipo:** E-commerce  
- **Principais fluxos testados:**
  - Cadastro de usuário
  - Login e Logout
  - Listagem e busca de produtos
  - Carrinho de compras
  - Checkout
  - Conta do usuário

---

## 🧪 Tipos de Testes

- Testes funcionais
- Testes de validação (positivos e negativos)
- Testes end-to-end (E2E)
- Smoke tests

---

## 🛠️ Tecnologias Utilizadas

- Python
- Playwright
- Pytest
- Git / GitHub

---

## ▶️ Como Executar os Testes


1. Instalar as dependências:
pip install -r requirements.txt

2. Instalar os navegadores do Playwright:
playwright install

3. Executar os testes:
pytest
